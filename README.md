## Smart Job Tracker

A Django-based web application to track job applications and manage statuses.

### Features
✔ Add and manage job applications  
✔ Update application statuses  
✔ Track company and role details  
✔ Simple and clean workflow  

### Tech Stack
- Python
- Django
- HTML / CSS

This project demonstrates practical backend development and application logic.
